#### 三个常用排序
1. 选择排序 

流程：
- 0 - N 上找最大的数与 N 位置交换
- 0 - N-1 上找最大的数与 N-1 位置交换
- 以此类推...
- 时间复杂度 O（N^2）
- 2，1（1），1（2），3 进行排序
- 2，1（1），1（2），3
- 1（1），1（2），2，3
- 1（2），1（1），2，3
- 所以选择排序是**不稳定**的

2. 冒泡排序

流程：
- 0 - N 以此比较相邻两个数字大小，并交换（0，1   1，2   2，3   ...）
- 0 - N-1 以此比较相邻两个数字大小，并交换
- 以此类推...
- 时间复杂度 O（N^2）
- **稳定**

3. 插入排序

流程：
- 保证 0 - 0 有序 --> 【（5），2，1，4，3】 
- 保证 0 - 1 有序 --> 【（2，5），1，4，3】
- 保证 0 - 2 有序 --> 【（1，2，5），4，3】
- 当前位置的数依次与左边的比较
- 若 i > i-1 不变，继续 i+1 与 i 比较
- 若 i < i-1 i 与 i-1 交换，继续 i-1 与 i-2 比较
- 时间复杂度 O（N^2）
- **稳定**
